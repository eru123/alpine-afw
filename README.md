# alpine-afw
Alpine custom image for Nodejs and Mongodb hosted in Azure VPS


## afw
```bash
docker pull lighty262/afw:latest
```
 - alpine:3.15
 - node:^16.x
 - npm:^8.x
 - mongodb:3

## alpine-npm
```bash
docker pull lighty262/alpine-npm:latest
```
 - alpine:3.15
 - node:^16.x
 - npm:^8.x

## alpine-mongodb
```bash
docker pull lighty262/alpine-mongodb:latest
```
 - alpine:3.15
 - mongodb:3
